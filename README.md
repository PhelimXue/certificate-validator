# Certificate Validator
***
### 此篇主要做兩個範例
* ValidateKeyAndCertificate.java
  * 驗證 private key 與 certificate 是否為一對。
* ValidateCATrust.java
  * 驗證 certificate 上下關係，也就是 CA 與 certificate 的關聯。
